# 🪪 3D Interactive ID Card Portfolio
</div>


## 🪪 About 

An immersive and interactive 3D digital ID card built using modern web technologies. Instead of a traditional portfolio website, this project presents personal and professional information through a realistic animated ID card experience which gets streched and played along if dragged.

## 🌐 Live Demo

🔗 https://3d-idcard.netlify.app/

---
## ✨ Features

- 🪪 Interactive 3D ID Card
- 🎭 Smooth Hover & Tilt Effects
- ⚡ Real-Time Animations
- 📱 Responsive Design
- 🌙 Modern Dark Theme
- 👨‍💻 Developer Information Display
- 🔗 Social Media Integration
- 🚀 Fast Loading & Optimized Performance

---


## 🛠️ Tech Stack

### Frontend

- React.js
- TypeScript
- Tailwind CSS

### 3D & Animation

- Three.js
- React Three Fiber
- Framer Motion

### Deployment

- Netlify

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Aaditya-Thk/id-3d.git
```


Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
